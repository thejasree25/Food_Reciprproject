
🍽️ Food Recipe Project

A modern React-based Recipe Application that allows users to discover delicious recipes from around the world. The application fetches recipe data from an external API and presents it in a clean, responsive, and user-friendly interface. Users can explore recipes, view ingredients, follow cooking instructions, switch between light and dark themes, and interact with dynamic UI features.

📖 Project Description

The Food Recipe Project is designed to help users discover and prepare delicious meals effortlessly. The application communicates with a Recipe API to retrieve recipe information and dynamically renders it on the user interface.

Each recipe card contains essential information such as:

Recipe Name
Recipe Image
Ingredients
Cooking Instructions
Category Information
Additional Recipe Details

Users can click on a recipe to view complete cooking instructions and ingredient lists required to prepare the dish.

✨ Key Functionalities
