---

## ✨ Overview

Food Recipe Finder is a frontend-only project built using **React + Vite**. It fetches recipe data from an external API and displays it in an interactive and user-friendly interface. Users can click on any recipe to view full details including ingredients and step-by-step cooking instructions.

---

## 🚀 Features

- 🔍 Fetch recipes dynamically from an external API  
- 🍽️ View detailed recipe information on click  
- 🍳 Step-by-step cooking instructions  
- 🌙 Dark / Light theme toggle  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- ⚡ Fast performance using Vite  

---

## 🛠️ Tech Stack

- React.js (Frontend Library)  
- Vite (Build Tool)  
- REST API (Data Source)  
- CSS3 (Styling)  
- JavaScript (Logic & Functionality)
